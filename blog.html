What is the Document Object Model (DOM)

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a webpage as a tree of objects, where each node corresponds to an element, attribute, or piece of text in the HTML document. The DOM allows JavaScript to access, modify, and interact with the content and structure of a webpage dynamically, making it essential for creating interactive web applications.

How do you select an element from the DOM?

You can select elements from the DOM using JavaScript in several ways, mainly through methods provided by document:

  • document.getElementById(id): Selects an element by its unique ID.
  • document.getElementsByClassName(className): Selects all elements with a specific class
  • document.getElementsByTagName(tagName): Selects all elements with a specific tag name.
    • What is event delegation in the context of the DOM, and why is it useful?

    Event delegation is a technique where instead of adding event listeners to individual child elements, a single event listener is added to a common parent element. This parent then listens for events from its children through event bubbling, where the event moves up the DOM tree from the target element to the parent. When an event occurs, you check the event’s target to determine which child element triggered it.

    How do you manipulate an element's attributes and styles using the DOM?

    You can use JavaScript to change an element’s attributes and styles dynamically:

  • setAttribute(attribute, value): Sets a specified attribute on an element.
  • getAttribute(attribute): Retrieves the value of a specified attribute
  • Direct Property Access: Some attributes can be directly modified, like element.id, element.src, etc.
